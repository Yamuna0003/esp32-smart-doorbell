# esp32-smart-doorbell
Dual mode IoT Smart Doorbell using ESP32 - CAM with Motion Detection and Telegram Alerts

🚀 Overview
Detects motion using PIR
Allows manual doorbell press
Captures image using ESP32-CAM
Sends image instantly via Telegram

🛠️ Hardware Used
ESP32-CAM
PIR Sensor
Push Button
LED (Flash)

⚙️ Features
Dual trigger (Button + PIR)
Real-time image alerts
WiFi-based IoT communication
Flash-assisted night capture
Anti-spam logic
